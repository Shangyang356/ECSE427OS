To compile the codes you should use command "make".
To run the codes, you need to run printer.c on one terminal and client.c on the other terminal.
To use printer, you should use command"./printer num" num is the how many slots.
To use client, you should use command"./cliend num2 num3" num2 is the client ID and num3 is the number of pages that the user wants to print.
